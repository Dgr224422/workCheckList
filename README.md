# workCheckList

Готовый Telegram-бот на **aiogram 3** для чек-листов сотрудников кинотеатра.

## Что умеет

### Для сотрудников
- Запускать выполнение чек-листа.
- Отмечать каждый пункт как `да/нет`.
- При ответе `нет` указывать обязательный комментарий.
- Сохранять отчет о выполнении.
- Просматривать свои последние отчеты.

### Для администратора
- Создавать чек-листы.
- Добавлять пункты в существующие чек-листы.
- Смотреть список чек-листов и количество пунктов.
- Просматривать последние отчеты сотрудников.

## Быстрый старт

1. Установите зависимости:
   ```bash
   pip install -e .
   ```

2. Создайте `.env` файл:
   ```env
   BOT_TOKEN=ваш_токен_бота
   ADMIN_IDS=123456789,987654321
   DB_PATH=checklists.db
   ```

3. Запустите бота:
   ```bash
   python -m bot.main
   ```

## Структура

- `bot/main.py` — точка входа.
- `bot/config.py` — загрузка конфигурации из `.env`.
- `bot/services/database.py` — SQLite слой и таблицы.
- `bot/handlers/common.py` — `/start` и общее меню.
- `bot/handlers/admin.py` — админ-команды.
- `bot/handlers/worker.py` — сценарий выполнения чек-листов сотрудником.
